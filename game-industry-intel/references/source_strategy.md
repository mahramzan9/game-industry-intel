# Source Strategy

## General Search Pattern

Use multiple query intents for each game:

- Basic info: `site:domain 游戏名 游戏库`, `site:domain 游戏名 专区`, `site:domain 游戏名 资料`
- Review: `site:domain 游戏名 评测`, `site:domain 游戏名 试玩`, `site:domain 游戏名 测评`
- Gameplay/system: `site:domain 游戏名 攻略`, `site:domain 游戏名 玩法`, `site:domain 游戏名 养成`
- Operations: `site:domain 游戏名 版本`, `site:domain 游戏名 活动`, `site:domain 游戏名 更新`
- Sentiment: `site:taptap.cn 游戏名 评价`, `site:taptap.cn 游戏名 评论`

Prefer pages with dates, authors, clear game names, and original content.

## Chinese Sources

### 17173

Best for online games, mobile games, new game tracking, rankings, tests, and version/news pages.

Useful signals:
- Game category, status, test/open beta timing
- Version activities and operation rhythm
- MMO/commercial game positioning
- Popularity/ranking context

### 3DM

Best for PC/console/single-player games, news, reviews, guides, and release information.

Useful signals:
- Gameplay genre and mechanics
- Review language around strengths/weaknesses
- System requirements, platform availability, release timing
- Guide pages for systems and progression

### 7k7k

Best for mini games, web games, lightweight games, and younger audience signals.

Useful signals:
- Casual gameplay, session length, child/teen orientation
- Simple controls, themes, and play loops
- Web/mini-game category fit

### 4399

Best for mobile games, mini games, web games, rankings, downloads, and youth-facing player signals.

Useful signals:
- Mobile/web category
- Casual or younger audience positioning
- Popularity and recommendation context
- Basic gameplay and tags

### Ali213

Best for PC/console games, mobile games, topics, downloads, guides, and game database pages.

Useful signals:
- Platform, release date, genre, publisher/developer
- Guides that reveal progression, combat, and systems
- Topic pages for franchise/world context

### Gamersky

Best for PC/console games, reviews, topics, guides, news, and editorial framing.

Useful signals:
- Gameplay strengths, combat feel, art style
- Player-facing selling points
- Version/news for large titles
- Review and guide evidence for system depth

### TapTap

Best for mobile games, ratings, version logs, official posts, and player sentiment.

Useful signals:
- Score and community sentiment
- Positive, neutral, negative, and controversial points
- Version update notes and event announcements
- Monetization complaints or praise

TapTap is often dynamic. Use browser access when static retrieval is incomplete.

## Overseas Supplements

Use overseas sources when the game is overseas-first, has limited Chinese coverage, or needs global context.

Recommended order:
1. Official site/store page for facts
2. Steam/console store for tags, reviews, release data, monetization/DLC
3. Major media reviews for gameplay/art/combat synthesis
4. Reddit/Steam reviews/community only for sentiment examples

Do not let overseas sources override domestic market framing unless the user asks for global analysis.
