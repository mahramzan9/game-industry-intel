# game-industry-intel

Reusable Codex skill for researching a game by name and producing a Markdown game intelligence summary.

## Contents

- `game-industry-intel/SKILL.md`: skill entrypoint
- `game-industry-intel/references/`: extraction schema, source strategy, and markdown template
- `game-industry-intel/scripts/collect_game_pages.py`: helper for generating search queries

## Usage

Place the `game-industry-intel/` folder into a Codex skills directory, or reference the folder directly from another workspace.
